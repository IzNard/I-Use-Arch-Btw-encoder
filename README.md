# I-Use-Arch-Btw-encoder
Encoding method that transforms a string of text into printable sequences of selected tokens from "i use arch btw" that can be decoded back into plain text strings.
# Requirements
python i guess
# Usage
Upon starting the program, choose whether you want to encode plaintext to base-i-use-arch-btw by pressing e or by decoding from base-i-use-arch-btw into plaintext
